# kinoticket backend
 
 Documentation repository: <https://gitlab.com/lumaghg/kino-backend-doku/-/tree/main>

install:
```
    npm install
```

build-command:
```
    npm run build
```

run
```
    npm run start
```

run in watch mode
```
    npm run devstart
```

build docker container
```
    docker build -t backend .
```

run docker container
```
    docker run --publish 8001:3001 backend
```
